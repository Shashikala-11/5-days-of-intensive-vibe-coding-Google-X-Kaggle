# 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/yourusername/5-Day-Intensive-Vibe-Coding.git
```

Navigate to the project

```bash
cd 5-Day-Intensive-Vibe-Coding
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
